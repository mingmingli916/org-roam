It consists of two parts:
- object classification
- object location


Object classification is similar to [[id:13ECC0CF-EC58-43F7-B285-A35596DAA488][Image Classification]].

Object location is normally achieved with [[id:9B7200B8-6161-4190-ABC7-4A68FE869575][Anchor Box]]es and [[id:23F20771-6FE3-4C3B-8E60-85EC5B4E8905][Linear Regression]].
The linear regression in computed on anchor boxes which base on feature map.
