#+title: Cache Thought

[[id:B9552B32-3B5D-427C-8B21-44F636747E67][Bridge Thought]]


When we query something frequently, we can cache them in higher read speed materials.
It can reduce query time or reduce the operations to get the query content.


We can also cache the content we want to edit and leave the original content unchanged.
It has the benefit of recovering easier.
