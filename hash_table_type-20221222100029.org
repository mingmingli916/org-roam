#+title: Hash Table Type

The printed representation of a hash table specifies its properties and
contents, like this:

#+begin_example
(make-hash-table)
;; #s(hash-table size 65 test eql rehash-size 1.5 rehash-threshold 0.8125 data ( ...))
#+end_example
